Private method initalize_steps

After all the steps prototypes have been created and mapped we begin the initialization process. This method contains two options only and except. You may only pass in one option to run over the resultant set of steps.

initialize_steps: (options={}) -> new ember.RSVP.Promise (resolve, reject) => @_warn "you passed in both 'only' and 'except' options to initialize_steps()" if ember.isPresent(options.only) and ember.isPresent(options.except) only = ember.makeArray options.only except = ember.makeArray options.except

Grab full set of steps and then run any optional filtering of the steps

steps = @get('steps') steps = @intersection(steps, only) if ember.isPresent(options.only) steps = @difference(steps, except) if ember.isPresent(options.except)

Iterate of each step and call its individual step initialization promise and push into an array to process the promises.

promises = [] steps.forEach (step) => promises.pushObject(step.initialize()) if step.initialize? ember.RSVP.Promise.all(promises).then => resolve()
